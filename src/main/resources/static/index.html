Welcome to the Text Transformer API

To apply a transformation you can either do a:

GET Request:

/transform?text=YOUR_TEXT&actions=YOUR_ACTION_1&actions=YOUR_ACTION_2

POST Request with payload:

{ "text": "YOUR_TEXT", "actions": [ "YOUR_ACTION_1", "YOUR_ACTION_2" ] }

Either way output will be in format:

{ "text": "YOUR_TEXT", "actions": [ "YOUR_ACTION_1", "YOUR_ACTION_2" ], "result": "YOUR_TEXT_AFTER_TRANSFORMATION" }
